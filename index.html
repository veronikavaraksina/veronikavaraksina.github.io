IBM dataset (RMarkdown)

This report is done using IBM Attrition Dataset. Researching and modelling attrition can be one of the ways to reduce IBM turnover rate. Not only will we be able to understand company’s “problem” zones when it comes to turnover, but we also will able to predict employees attrition. That information can help HR department make data driven decisions and build new strategy for reducing turnover rate, helping company retain competent employees and reduce HR costs.

IKEA dataset (RMarkdown)

In this report I will cover the data analysis of the IKEA online store data set. This data set being used for this research has been created through web scraping of the IKEA Saudi Arabian website in 2020. The aim of this report is to answer business decision-making questions by utilizing the data set: What affects the price of furniture? Based on what factors IKEA furniture is on discount? How to predict the category of the furniture?

NLP report (RMarkdown)

This analysis and insights will assist Disney+ marketing department to better position promotion for upcoming shows. The report was completed by collecting tweets with relevant hashtag and processing them using NLP tools.

NLP Depression dataset (Python)

This report was done using Python and NLP packages. The dataset consists of more than thousand tweets. Based on the sentiment analysis, the report predicts if tweet has depressive topic within it or not.

Insurance dataset (Python)
